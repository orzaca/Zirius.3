IMAGEN PIXELEADA O CONGELADA

Acciones:

1- Verificar que el STB este despejado sin trapos u objetos encima del STB o que no este sobre la TV

2- Confirmar que el STB este conectado y encendido

3. Ejecutar el comando Cold Init *No usar en STB con DVR*

¿Servicio OK?

Acciones:

1- Quite las baterias del control y vuelva a colocarlas

2- Confirma que permita cambiar de canal con el control, o ingrese al menu del STB

3- ejecutar el comando Cold Init *No usar en STB con DVR*

¿Servicio OK?

Revisar si los canales pertenecen a la grilla contratada

1- Si no esta en el listado de su plan, informarle al cliente

2- Verifica si no hay canales bloqueados

Los canales son del servicio Basico?

Acciones

1- Ejecutar el comando NEW_ADDPACKETS_HFC

2. Si no funciona, ejecutar el comando REFRESH_HFC

¿Ya tiene Señal?

Verifica si no tiene WF abiertos, en caso tal realizar seguimiento a orden

¿Que mensaje le aparace en el canal?

"PARA CONTRATAR A ESTE PAQUETE MARQUE 24201515 O 147-100"

1- Ejecutar el comando NEW_ADDPACKETS_HFC

2. Si no funciona, ejecutar el comando REFRESH_HFC

"TEMPORALMENTE SIN SEÑAL" o la pantalla de color negro

1. Confirmar que el STB este encendido

2- Ejecutar el comando REFRESH_HFC

2. Si no funciona, ejecutar el comando NEW_ADDPACKETS_HFC

3. Si no funciona, ejecutar el comando Cold Init *No usar en STB con DVR*

3. Si no funciona, ejecutar el comando Reinstall_Setup_HFc *No usar en STB con DVR*

¿Tiene Señal?

Verificar Conexiones

1. Desconecte y conecte el cable coaxial de la tv y de la cajita e invertir las puntas

2. Confirme en que opcion esta configurado, puede ser canal 3 o canal 4

¿Ya tiene Señal?

Acciones

1. Confirmar que el cable coaxial este bien apretado.

2. Desconectar y Conectar el Router

3. Esperar que sincronicen las luces

4. Confirma Navegacion aun si no haya cambiado el estado del puerto en axiros

¿Cliente Puede Navegar?

Acciones

1. Solicitar al cliente que le de "Olvidar Red" a su red Wifi.

2. Validar en UMP El nombre de la red Wifi y confirmar con el cliente si le muestra.

3. Si está en OFF, activa la red Wifi.

¿Cliente Puede Ver la red WIFI?

ACCIONES:

1. Indicale al cliente que se Conecte a la red WIFI.

Cliente se conecta y Navega?:

Cambiar Nombre de Red y Contraseña:

1. Debes realizar cambio de contraseña y nombre de red en UMP.

2. Debes Validar la titularidad (Ingresa pagina restricciones).

3. Guardas los cambios y refrescas.

Cliente se conecta y Navega?:

Acciones

1. Indicarle al cliente que verifique si esta conectado el cable tanto en la PC como en el router.

2. Desconectar y conectar el cable de red

3. Deshabilitar y Habilitar la tarjeta de red.
