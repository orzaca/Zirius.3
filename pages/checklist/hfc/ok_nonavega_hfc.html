¿Resultado 100%?

¿Cliente llama del mismo Numero?

1. Si llama del mismo numero: Pedir un numero movil y devolver la llamada.

2. Enviar REBOOT desde Axiros.

3. Esperar que sincronicen las luces.

4. Confirma si cliente puede navegar.

¿Cliente logra navegar?

¿Cliente llama del mismo Numero?

1. Si llama del mismo numero: Pedir un numero movil y devolver la llamada.

2. Enviar REBOOT desde Axiros.

3. Si esta por via Wifi, indicarle que se acerque lo mas que pueda al router.

4. Debe desconectar los demas dispositivos.

4. Confirma si cliente puede navegar.

¿Cliente logra navegar?

Acciones

1. Confirma que cliente no tenga IP publica, si la tiene omite este paso.

2. Indicar al cliente si tiene alguna aguja o un clip.

3. Indicarle que inserte la aguja en un orificio llamado reset en el router por 10 segundos

4. Las luces del router se apagaran, esperar que sincronicen nuevamente las luces

5. Confirma si cliente puede navegar

¿Cliente logra navegar

Preguntar al Cliente por que medio esta realizando las pruebas
